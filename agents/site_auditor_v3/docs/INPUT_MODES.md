# INPUT_MODES

LINK audit = public link/surface audit.

ZIP audit = ZIP/package audit + LINK audit when live target exists.

REPO audit = repository audit + LINK audit when live target exists.

PROMPT/THEMATIC audit = LINK audit with custom rubric/lens.
It is not essay mode.

If live target cannot be resolved in ZIP/REPO mode:
status must be PARTIAL, not fake PASS.
