# PRODUCT_MEMORY — SITE_AUDITOR_V3

SITE_AUDITOR_V3 is a Universal Audit Engine.

Core loop:
input -> weak points -> evidence -> decision -> action -> next capability

It is not:
- crawler
- link checker
- screenshot bot
- SEO scanner
- report generator

It must:
- find weak points
- prove them with evidence
- decide one strongest action
- declare limits
- diagnose failures
- show what capability to build next

Forbidden:
- silent fail
- fake PASS
- PASS with weak evidence
- giant runtime
- hardcoded orchestration
- output inventing findings
