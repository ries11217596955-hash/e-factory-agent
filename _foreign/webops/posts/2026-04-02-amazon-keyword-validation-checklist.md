# Amazon Keyword Validation Checklist

Validation prevents keyword stuffing and weak listing edits.

## Checklist
- Is the keyword relevant to the actual product?
- Does it reflect buyer intent?
- Can it be placed naturally in the listing?
- Does it duplicate a stronger phrase?
- Does it improve coverage without harming clarity?

## Final rule
Keep only terms that support both discovery and comprehension.
