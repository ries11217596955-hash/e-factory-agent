# Shopify abandoned cart recovery map

Map the abandoned-cart path before adding more tools.

## Step 1: Split the problem

Separate:
- casual browsers
- price-sensitive visitors
- distracted repeat buyers
- customers blocked by product-page uncertainty

## Step 2: Build the ladder

Use a simple 3-touch flow:
1. reminder
2. objection remover
3. offer or urgency

## Step 3: Add signals

Track:
- cart value
- return visitor
- viewed products
- coupon usage

## Step 4: Protect the sequence

Do not:
- spam daily
- mix too many offers
- send discount first
