# Shopify return request triage

A return request should not become a long manual chain.

## Triage lanes

- auto-safe
- review-needed
- escalate

## Minimum fields

- order number
- product
- reason
- delivery status
- refund or exchange request

## Why this matters

Good triage reduces:
- handling time
- repeat replies
- refund confusion
- hidden defect patterns
