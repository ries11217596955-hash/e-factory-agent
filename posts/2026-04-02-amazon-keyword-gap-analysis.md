# Amazon Keyword Gap Analysis

Gap analysis shows which relevant terms competitors rank for but your listing misses.

## What to compare
- top competitor title
- bullet points
- backend search terms
- recurring review language

## Process
1. Pick 3-5 direct competitors.
2. Extract visible keyword patterns.
3. Mark missing terms in your own listing.
4. Rank gaps by relevance and search intent.
5. Add only the terms that improve clarity and match the product truth.
